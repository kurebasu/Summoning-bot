circles:
All files in circles should be named <magic circle number.gif>, an example would be "magic circle 1.gif"
These files should be numbered from 1 to n, where n is the amount of images you want to use.

perry:
All files in perry should be named <perryNumber.gif>, an example would be "perry1.gif"
These files should be numbered from 1 to n, where n is the amount of images you want to use.

specific:
All files in specific should be named <triggerword.gif>, an example world be "eat.gif"
When the trigger word is used in a message, the bot will use the specific gif.
